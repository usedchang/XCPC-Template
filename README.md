# LaTeX CP 模板合集 🚀

我的个人 LaTeX 竞赛编程模板仓库（基于 TeX Live + VS Code）。

## 📁 包含内容
- `main.tex`：主模板（已集成常用宏包、页眉页脚、算法环境）
- 主席树、树状数组、ST表 等模板即将上线
- `.gitignore`：自动忽略所有 PDF 文件

## 🚀 使用方法
1. 下载 `main.tex`
2. 用 VS Code + LaTeX Workshop 打开
3. `Ctrl+Alt+B` 一键编译
